# Semiconductor Test Support Lab

This project simulates a basic Application Support Engineering environment inspired by semiconductor test automation workflows.

The goal of this lab is to practice Linux administration, Git, GitHub, Python scripting, Jenkins pipelines, PostgreSQL, Nginx, performance benchmarking, monitoring, and troubleshooting documentation.

## Lab Architecture

Windows Workstation  
↓ SSH / Browser / Git  
Ubuntu Server VM  
↓  
Python Test Simulator  
↓  
Jenkins Pipeline  
↓  
PostgreSQL Database  
↓  
Nginx Dashboard
